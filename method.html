Методы это...




Метод - это действие/команда которую выполняет компьютер.

Метод задаётся таким образом:

Имя_метод (Параметры)
Параметры мы изучим в следующем уроке


Например в Python метод Print() выводит значения на кослоль.

КОД:


Print ('Hello')




***************************************************
Эта строчка выведет строчку hello

Print - метод
'Hello' - параметр
