Параметры это...




Параметры это данные для методов.

Рассмотрим пример

КОД:


Print ('Hello')




***************************************************


В данном случае метод Print использует параметр 'Hello' что-бы вывести строку Hello
Одинарные кавычки используются для обозначения строки. Цифры пишутся без запятой

Для этого и нужны параметры.

В разных методах разное кол-во параметров.

Параметры пишутся через запятую.

Например в Print() их неограниченное параметров.


КОД:


Print ('Hello World','!')
